# Ekklesia-backend

## Criação do ambiente virtual

Windows
```ps1
python -m venv .venv
```

Linux
```bash
sudo apt install python3-venv
python -m venv .venv
```

## Ativar o ambiente virtual

Windows
```ps1
.\.venv\Scripts\activate
```

Linux
```bash
source .venv/bin/activate
```

## Instalar as dependências

```bash
pip install -r requirements.txt
```
